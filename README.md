## 📁 Featured Projects

### 🤖 [Gesture Controlled Virtual Telepresence Robot](https://github.com/mani-minion/gesture-controlled-telepresence-robot)
> **Published at ICICAEPHAM 2023** | Paper ID: ICIC2AEPHAM230164

A four-wheeled telepresence robot controlled by **hand gestures and head movements** using MPU6050 IMU sensors, ESP32, and Raspberry Pi. Live VR video streamed to a smartphone inside a VR headset over Wi-Fi/WebSocket.

`Python` `ESP32` `Raspberry Pi` `MPU6050` `WebSocket` `Ubuntu` `Arduino IDE`

---

### 🦾 [SCARA Robot for Vision Inspection](https://github.com/manikanda-prabhu-m/scara-robot-vision-inspection)

A 3-axis SCARA robot integrated with a **TensorFlow 2 SSD MobileNet V2** object detection pipeline deployed on Raspberry Pi 4. Inverse kinematics solved using Tinyik. Two-mode inspection: continuous conveyor checking and batchwise SCARA-assisted internal inspection.

`Python` `TensorFlow 2` `SSD MobileNet V2` `Tinyik` `Raspberry Pi 4` `PCA9685` `Robot Kinematics`

---

### 🗺️ [ROS Mapping & Autonomous Navigation](https://github.com/mani-minion/ros-mapping-navigation)
> Based on RigBetel Labs workshop certification (Apr 2022)

ROS implementation of **SLAM-based mapping** (GMapping) and **autonomous navigation** (move_base) with a differential drive robot in Gazebo. Includes Publisher/Subscriber examples and programmatic goal sending.

`ROS` `Gazebo` `RViz` `GMapping` `SLAM` `URDF` `Navigation Stack`

---

### 🧭 [Autonomous Mobile Robot — Core Algorithms](https://github.com/mani-minion/autonomous-mobile-robot-basics)
Implementations of fundamental AMR algorithms from scratch:
- **A\* Path Planning** — optimal grid-based path search with visualization
- **Particle Filter (MCL)** — Monte Carlo Localization for probabilistic robot localization

`Python` `NumPy` `Matplotlib` `Probabilistic Robotics` `Path Planning`

---

### 🏭 [Bottle Label Detection using PLC & Vision Sensor](https://github.com/mani-minion/bottle-label-detection-plc)
Industrial quality inspection system using **SIEMENS PLC** (ladder diagram), **Banner Q3X laser contrast sensor**, and **SCADA** to detect missing labels on conveyor belts.

`SIEMENS PLC` `Ladder Diagram` `SCADA` `Python Simulation`
